Projects

MDVibCor – a program for computing vibrational corrections for gas-phase electron diffraction using molecular dynamics data. Written in C++ with the usage of STL and GSL (console application for Linux and Windows). The initial development was in Edinburgh in 2008 (method description is published in [14]), then in Bielefeld (see ref. [1]). In 2013 the project has been placed on GitHub (project page) under GPLv3 license.

Z2IDP – a program for data reduction of gas-phase electron diffraction images obtained by scanning diffraction patterns. Written in C++ with the use of Qt (Linux and Windows versions). Method description is given in ref. [6, 11]). Previous variants of this software, XIDP (C++ and wxWidgets, Linux and Windows versions) and IDP (Delphi, Windows) were designed for processing data from the MD-100 microdensitometer (method description can be found in ref. [21]).

A set of utilities for simplifying computations of vibrational corrections using the SHRINK program (Q2SHRINK – input file generation, SHREx – data extraction from output files, ExPED – extraction of potential energy distributions). Written in Python, the description is in ref. [13].

Personal website, which you are reading (HTML, CSS3)

MD100 – A program for control and data acquisition for the MD-100 automated microdensitometer. Written in Borland Pascal, has been used for scanning diffraction patterns in the ISUCT laboratory of gas-phase electron diffraction for 15 years. The description of the microdensitometer and the technique is published in [21, 29]. It became obsolete due to transition to scanning diffraction patterns using a commercial scanner.

AZ's Book Reader – a program for reading e-books (probably one of the first). It waswritten in 1997 in Delphi, distributed as freeware, and supported only ASCII texts (in several encodings) and working with archives (mainly for texts distributed through Fido newsgroups). It became completely obsolete with the appearance of CoolReader, FBReader, FictionBook format, etc., the last version was released in 2000. Curiously enough, it is still available for downloading in the Internet.
