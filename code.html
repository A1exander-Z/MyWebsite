Я пишу программы уже более 20 лет, начав с ZX Spectrum (Sinclair BASIC и машинные коды Z80), а затем перейдя к Borland Pascal на PC. В середине 90-х я освоил Object Pascal/Delphi, на котором написаны мои первые серьезные проекты, а в начале 2000-х заинтересовался другими языками, в первую очередь C/C++ и Python, которые и стали моими основными средствами разработки. Среди моих интересов – высокопроизводительные вычисления и машинное обучение, в том числе нейронные сети.

Проекты

MDVibCor – программа для расчетов колебательных поправок для газовой электронографии по данным молекулярной динамики. Написана на C++ с использованием STL и GSL (консольное приложение для Linux и Windows). Первоначальная разработка велась в Эдинбурге в 2008 г. (описание методики опубликовано в [14]), затем в Билефельде (см. работу [1]). В 2013 г. проект размещен на GitHub (страница проекта) под лицензией GPLv3.

Z2IDP – программа первичной обработки электронографических данных, полученных при сканировании электронограмм. Написана на C++ с использованием Qt (версии для Linux и Windows). Описание методики приведено в работах [6, 11]). Предыдущие варианты данной программы, XIDP (C++ и wxWidgets, версии для Linux и Windows) и IDP (Delphi, Windows) были предназначены для обработки данных с микроденситометра MD-100 (описание методики см. в [21]).

Набор программ для облегчения расчета колебательных поправок с помощью программы SHRINK (Q2SHRINK – генерация входных файлов, SHREx – извлечение данных из выходных файлов, ExPED – извлечение распределения потенциальной энергии). Написаны на Python, описание см. в [13].

Персональный сайт, который вы сейчас читаете (HTML, CSS3)

MD100 – программа управления и сбора данных для автоматизированного микроденситометра MD-100. Написана на Borland Pascal под MS-DOS, использовалась для фотометрирования электронограмм в лаборатории газовой электронографии ИГХТУ в течение 15 лет. Описание микроденситометра и методики опубликовано в работах [21, 29]). Устарела в связи с переходом на сканирование электронограмм.

AZ's Book Reader – программа для чтения книг с экрана компьютера (вероятно, одна из первых). Написана в 1997 г. на Delphi, распространялась как freeware, поддерживала только тексты в формате ASCII (в различных кодировках) и работу с архивами (в первую очередь, для текстов, распространявшихся через эхоконференции Fido). C появлением CoolReader, FBReader, формата FictionBook и т.д. полностью устарела, последняя версия относится к 2000 г. Как ни странно, до сих пор доступна для скачивания в Интернете.
